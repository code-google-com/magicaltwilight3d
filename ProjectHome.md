# 3D SDK and Game #

This Project shows that it is possible to write a 3D RPG in the relatively new programming language purebasic.

  * 3D RPG GAME
  * Items
  * Enemys
  * 3rd and 1st person switch
  * Shader
  * N3xtD 3D Engine
  * 3D Sound Engine (irrklang)
  * Physics Engine (Newton)
  * and much more
